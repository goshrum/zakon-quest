# Support

Thanks for using **Know the Law**! Here is how to get help.

- **Questions & ideas:** open a
  [GitHub Discussion](https://github.com/goshrum/zakon-quest/discussions).
- **Bugs:** open an issue using the **Bug report** template.
- **Feature requests:** open an issue using the **Feature request** template.
- **Wrong or outdated legal content:** open an issue using the
  **Question correction** template and include a citation/source.
- **Security issues:** see [`SECURITY.md`](SECURITY.md) — do **not** file a
  public issue.

Before opening an issue, please search existing issues and discussions to avoid
duplicates. For contribution guidelines see [`CONTRIBUTING.md`](CONTRIBUTING.md).
